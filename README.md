# Winglet 🐦

Winglet is a simple 2D browser game inspired by Flappy Bird, built as a learning project using HTML, CSS, and JavaScript.

The goal of this project is to understand game loops, canvas rendering, keyboard input handling, and basic game state management.

---

## 👀 Preview

A lightweight Flappy Bird-inspired browser game built using the HTML Canvas API.

---

## 🎮 Gameplay

- Press **Space** to start the game
- Press **Space** to make the bird jump
- Avoid the pipes
- Each successful pass increases your score
- Game restarts after game over

---
